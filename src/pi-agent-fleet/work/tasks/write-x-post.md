# Writing Task: X Post from Research Brief

## Goal

Turn the attached AI engineering research brief into one publish-ready X post for ZazenCodes.

## Audience

Software engineers learning how to use coding agents, agentic workflows, and modern AI engineering tools.

## Inputs

- The research brief attached to this run (latest entry in `work/outputs/research-ai-engineering.md`).
- ZazenCodes voice: practical agentic coding, first-person, no hype.

## Output Format

Write a concise Markdown draft with:

- One primary post (under 280 characters, ready to paste)
- One alternate version with a different hook
- The source link
- The next best specialist agent to run

## Constraints

- Use only facts from the brief. Do not invent stats, quotes, or freshness.
- No hashtag spam (one hashtag max, or none).
- No engagement bait or hype words ("game-changer", "insane", "🚀").
- Keep the draft useful for a downstream comment-reply agent.
