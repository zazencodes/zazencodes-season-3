# Research Task: AI Engineering Trends

## Goal

Find 3 timely AI engineering or coding-agent stories that could become a ZazenCodes X post or short video angle.

## Audience

Software engineers learning how to use coding agents, agentic workflows, and modern AI engineering tools.

## Inputs

- ZazenCodes focuses on practical agentic coding.
- The content should help engineers become better at using AI in real development work.
- Prioritize stories that connect to hands-on workflows.

## Output Format

Write a concise Markdown research brief with:

- 3 candidate stories
- Why each story matters
- A ZazenCodes content angle for each
- Source links
- One recommended pick
- The next best specialist agent to run

## Constraints

- Prefer primary sources.
- Only include stories with source links fetched in this run.
- Do not invent freshness, popularity, benchmark scores, or source statistics.
- Do not use generic AI hype.
- Keep the brief useful for a downstream social post writer.
